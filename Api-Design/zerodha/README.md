To generate docs --<br>
```
cmd: pwd
out: D:\Java\zerodha

cmd: ls
out:
zerodha (directory)
Main.java
README.md
// ...more

cmd: mkdir docs
out:

cmd: javadoc -d D:\Java\zerodha\docs .\zerodha\*.java
out: The above command generates Java documentation for all Java files in the 'zerodha' directory 
     and saves it in the 'docs' directory.
```
